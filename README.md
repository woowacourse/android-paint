## 1단계: 자유 곡선

- 그림판 앱을 구현한다.

### 기능 요구 사항 🛠️

- 화면에 자유 곡선을 그릴 수 있다.
- 도구 패널을 추가한다.
    - 브러시 색상을 변경할 수 있다. (브러시 색상은 빨, 주, 노, 초, 파로 한정)
    - 브러시 굵기를 변경할 수 있다.
